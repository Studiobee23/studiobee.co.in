Insight

→

Dark sections + tight −0.04em tracking signal premium positioning the instant the page loads. Work section placed second — not behind a click.
Insight
Indian founders prioritise warmth and approachability alongside prestige — cold, editorial-only aesthetics feel inaccessible to the target market.
→
Decision
Cream section for Team + Kulim Park body font — breaks the dark/blue cadence deliberately. Warm contrast is a brand signal, not a deviation.
Constraint
iPhone 12/13 crashes Chrome with more than ~50 GPU compositor layers — each animated clip-path element consumes one.
→
Decision
Hard layer budget of 50: 20 hero strips + 10 text inversion + 10 nav inversion + 10 hamburger inversion = exactly at limit. No decorative animations beyond this.
Constraint
The contact form is the only conversion point — it must feel like a destination, not a form at the bottom of a page.
→
Decision
Full-section blue background for CTA — matches the brand's primary blue. The section acts as a visual full-stop. Decorated success state with SVG checkmark + countdown bar makes submission a reward.
Constraint
Nav must work at 390px without collapsing to microscopic tap targets — WCAG 2.5.5 requires 44px minimum.
→
Decision
Full-screen dark overlay menu on mobile — large display-size link text. What could feel like a limitation became a brand-scale interaction.
Hero — Wave Animation System05 / 10
Hero Design
A motion system born from brand energy — not decoration.
Wave Strip System
How It Works
20 HTML divs with identical clip-path shapes. Each animates its opacity on a 9-second sinusoidal wave — staggered by 0.45s per strip — creating a left-to-right travelling shimmer. Static clip-path (no clip-path animation) keeps each strip to exactly 1 GPU compositor layer.
iOS Performance Design
Animating clip-path itself would double the layer count. By keeping clip-path static and only animating opacity, each strip stays GPU-composited without triggering layout or paint.
Text Inversion System
Headings appear to change colour as strips pass through them — achieved via 3 separate inversion layers (text, nav logo, hamburger), each with 10 matching strip clones. Same animation, exact same coordinates = perfect synchronisation at zero extra CPU cost.
Off-Screen Pause
All 50 strip animations pause via IntersectionObserver when the hero scrolls out of view — eliminating battery drain mid-scroll through the rest of the page.
Information Architecture06 / 10
IA & Conversion Architecture
Section order is as intentional as section design.
Every section position is a deliberate conversion decision. The visitor arrives at the Hero, sees proof of work immediately (Work is second — not behind a click), understands process, and reaches the CTA in 3 scroll stops.
Why Work is Section #2
Proof before pitch
Showing work before explaining services bypasses the credibility question. The visitor validates quality in their first 8 seconds — then they're primed to read how you do it.
Fixed Navigation + Pill CTA
Text links navigate, buttons act
'Start a project' is the only pill button in the nav — creating a grammar: all other items are navigation; this one is a conversion action. Always visible. Nav gains blur background on scroll to signal context shift.
Work Section — Auto-scroll Marquee
Show more without asking for a click
A horizontal auto-scroll marquee surfaces the entire portfolio passively. Cards are 9:16 portrait — Instagram-native ratio. The user sees quality breadth without taking any action. RAF-cancelled when section scrolls off-screen.

Hard Rule
No mix-blend-mode on fixed full-viewport elements. No backdrop-filter on carousel cards. No animated clip-path — only animate opacity and transform.
Architecture Decisions
No Framework
Single HTML file, inline CSS, vanilla JS. Zero build step, zero dependency risk. Full control over what gets painted and when. The entire site loads in one HTTP request.
RAF Throttling
Nav scroll handler runs inside requestAnimationFrame — one DOM read per frame maximum. Carousel cancelAnimationFrames completely when scrolled off-screen.
IntersectionObserver
Hero wave strips pause when scrolled out of view. Work marquee pauses when the section is invisible. Text scramble triggers once on first intersection. Zero wasted computation off-screen.
Video Preload Strategy
preload="metadata" (not "none") ensures first frames are decoded for autoplay — avoiding the blank-card flash while still deferring full download.

Design Principles That Held
Constraints → Creativity
The iOS 50-layer limit forced a systematic approach to animation. That constraint produced a more coherent motion language — every animated element had to earn its layer. The result is more disciplined than an unconstrained design would have been.
Restraint is a skill
The hardest decisions were removals. A mid-scroll parallax effect was cut because it added a layer. A decorative grain overlay was cut because it conflicted with the inversion system. Every cut made the remaining elements stronger.
Build = Design Continued
Implementing the design surfaced decisions that Figma couldn't — exact animation easing, scroll physics, the precise timing that makes the text scramble feel snappy not jittery. The design and build were one continuous process, not two handoffs.
Brand as System
Defining the brand as tokens first (colours, type scales, section cadence) rather than starting with layouts meant every new section decision was guided — not invented. The system made design faster and more consistent.
